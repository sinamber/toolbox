## Getting Started
### Prerequisites
* nodejs
* yarn or npm
* rust
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sinamber/toolbox
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. just run for dev test
   ```sh
   https://github.com/sinamber/toolbox
   ```
4. run for tauri dev test
   ```sh
   yarn tauri dev
   ```
5. build toolbox
   ```sh
   yarn tauri build
   ```
